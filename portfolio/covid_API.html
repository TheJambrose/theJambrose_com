CDC API: Compare States

As of today, COVID-19 continues to be an important issue that many people continue to stay up to date on. Making use of a public API (made available for free by the CDC), you can pull a report for the running total of deaths related COVID-19 for each US state.

Per the dataset details, this information is updated twice a daily. I've found this to be generally correct, though daily does not appear to include weekends as of right now.

Choose Two States to Compare

This is my first successful attempt at using an API to return data that could actually be of use to someone—even if that someone is only myself. You can review the full details of this dataset on the CDC's website.
